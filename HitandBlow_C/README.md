# Hit & Blow
ヒットアンドブローという数字当てゲームです。

## 概要
コンピューターが決めた3桁の数字を当てるゲームです。  

## ルール
- 百の位、十の位、一の位にそれぞれ数字を入力します。
- 数字と位置一致したらイート(eat)、数字のみ一致したらバイト(bite)として、それぞれの個数がわかります。
- 数字の重複は除きます。
